Zadanie 5:

Masz dane dwie tabele o następującym schemacie:

CREATE TABLE STATION_A	
(	STATION_ID NUMBER(*,0))

CREATE TABLE STATION_B
(	STATION_ID NUMBER(*,0))

Wskaż zapytania zwracające tylko STATION_ID występujące w obu tabelach (część wspólna podanych tabel):
Choose the right answers


SELECT A.STATION_ID
FROM STATION_A A
JOIN STATION_B B
ON A.STATION_ID = B.STATION_ID


SELECT A.STATION_ID
FROM STATION_A A
UNION
SELECT B.STATION_ID
FROM STATION_B B


SELECT A.STATION_ID
FROM STATION_A A
FULL OUTER JOIN STATION_B B
ON A.STATION_ID = B.STATION_ID


SELECT A.STATION_ID
FROM STATION_A A
INTERSECT
SELECT B.STATION_ID
FROM STATION_B B


SELECT A.STATION_ID
FROM STATION_A A
LEFT JOIN STATION_B B
ON A.STATION_ID = B.STATION_ID
WHERE B.STATION_ID IS NOT NULL
